## Como utilizar
### Configuração
#### message-service
```
cd .\backend\message-service
npm install
npm run migrate
```
#### user-service
```
cd .\backend\user-service
npm install
npm run migrate
```
#### frontend
```
cd ..\..\frontend\chat-frontend
npm install
```
### Como inicializar a execução
#### message-service
```
cd backend\message-service
npm run dev
```
#### user-service
```
cd backend\user-service
npm run dev
```
#### frontend
```
cd \frontend\chat-frontend
npm run dev
```
