## Requisitos

- Node.js - v22.6.0
- npm - 10.8.2
- Extensões (Visual Studio Code) - DotENV, Prisma

## Guia de execução

**2. Navegar até o Diretório do Projeto**

```bash
cd message-service
```

**3. Instalar Dependências**

```bash
npm install
```

**4. Executar migrações do Prisma**

```bash
npm run migrate
```

**5. Iniciar o Servidor de Desenvolvimento**

```bash
npm run dev
```
